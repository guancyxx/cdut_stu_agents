## 备份与恢复（仅题库，不含账号）

### 备份
```bash
# 仅导出题目相关表
docker exec cdut-postgres pg_dump -U cdut -d cdut_oj --clean --if-exists \
  -t public.problem -t public.problem_tag -t public.problem_tags \
  > /tmp/cdut_problem_schema_data.sql
```

### 恢复
```bash
docker exec -i cdut-postgres psql -U cdut -d cdut_oj < /tmp/cdut_problem_schema_data.sql
```

## 常见问题

### 1) 提交报 Sandbox unavailable
```bash
docker logs --tail 100 cdut-sandbox
curl -sS http://127.0.0.1:8899/health
```

### 2) 管理页面接口 404/403
```bash
# 确认已登录管理员
curl -sS -c /tmp/cookie.txt -H 'Content-Type: application/json' \
  -d '{"username":"admin","password":"cdut_oj_2026"}' \
  http://127.0.0.1:8850/api/login

curl -sS -b /tmp/cookie.txt http://127.0.0.1:8850/api/profile
```

### 3) 题目列表为空
- 检查 `public.problem` 是否有数据
- 检查前端筛选条件是否过严
